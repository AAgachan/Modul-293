Flappy Bird - Project

I personally found programming with Windows Forms to be the most enjoyable, so I decided to deepen my skills in it. As a result, I chose to develop a Flappy Bird game as my project. The game will offer a variety of new and unique levels that can be unlocked using keys. Players will have the option to purchase these keys once they achieve a sufficiently high score. My aim is to create a fully functional game while simultaneously expanding my understanding of Windows Forms.

Throughout the project, I anticipate gaining valuable experience in game development, user interface design, and Windows Forms programming. Ultimately, I hope to deliver an engaging and polished game that showcases my proficiency with Windows Forms.

  • Using controls such as buttons, labels and text fields in Windows Forms C#.
  • Use of controls such as buttons, labels and text boxes in Windows Forms.
  • Design of a user interface and arrangement of controls
  • Implementing functionalities such as setting ships and firing shots in a Battleship game
  • Using event handlers and properties of controls to implement desired functions

  • Creating and connecting different forms to create an interactive user interface.
  • Experimenting with the possibilities of the form connection and testing the functionalities

  • The coordination and interaction of the various Forms and their elements.
  • The understanding of the methods and commands for connecting Forms and updating the content.
