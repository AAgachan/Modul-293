PowerShell Script - Project

This project focuses on developing a script to automate tasks on a laptop. The main objective is to create a script that can efficiently organize files with specific extensions (.docx, .pptx, and .xlsx) into separate subfolders. Additionally, the script should have the capability to reset the sorting if needed. The script will be designed to recognize files with the specified extensions and move them to their corresponding subfolders.

This organization system will facilitate easier access and management of files, enhancing productivity and reducing clutter. By automating the sorting process, the script aims to save time and effort for the user, eliminating the need for manual file organization. It provides a convenient solution for individuals dealing with a large number of files in the specified formats.

Furthermore, the script will be designed to be user-friendly, ensuring that even non-technical users can easily operate it. Clear instructions and prompts will guide the user through the sorting and resetting process.

Overall, this project aims to simplify file management on a laptop by automating the sorting of specific file types into dedicated subfolders. The script's functionality and user-friendly design will contribute to a more organized and efficient workflow for users.

  • Creating a sorting and resetting script for files .
  • Reading file extensions and moving them to appropriate destination folders
  • Adding additional sorting criteria and checking paths
