Battleship Game - Project

In this project, I am focusing on programming the childhood game "Battleship" using Windows Forms. I aim to recreate the classic gameplay experience and make it accessible on a computer. The project involves designing a user-friendly interface with a grid-based layout to represent the game board. I will implement functionality for placing ships, guessing coordinates, and keeping track of hits and misses.

Additionally, I plan to incorporate features like multiplayer support and a computer opponent for solo play. Throughout the project, I will deepen my understanding of Windows Forms programming and enhance my problem-solving skills. Ultimately, I aspire to create a fully functional and enjoyable version of the "Battleship" game using Windows Forms.

  • Opening and closing windows with WinForms in C#.
  • Use of controls such as buttons, labels and text boxes in Windows Forms.
  • Using controls like buttons, labels and text boxes in Windows Forms.
  • Implementing functionalities such as setting ships and firing shots in a Battleship game
  • Using event handlers and properties of controls to implement desired functions

  • The creative design of the user interface with the various controls.
  • The implementation of the functionalities and the testing of the game

  • Getting started with WinForms and understanding the structure of Windows Forms.
  • The implementation of the desired functionalities with the right event handlers and properties.
